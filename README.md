# Big Mart Sales Project

## Project Overview
This project was completed in 5 levels using Power BI.

### Level 1 - Data Understanding & Import
- Imported Big Mart Sales dataset
- Explored columns and data types

### Level 2 - Data Cleaning & Preparation
- Handled missing values
- Standardized Fat Content values
- Renamed columns

### Level 3 - Basic Visualizations
- Created Bar Chart
- Column Chart
- Pie Chart
- Table Visual

### Level 4 - Interactivity & Insights
- KPI Cards
- Slicers
- Interactive Dashboard

### Level 5 - Final Dashboard
- Professional dashboard design
- Business insights
- Final project presentation

## Tools Used
- Power BI
- Power Query
- DAX
- GitHub
